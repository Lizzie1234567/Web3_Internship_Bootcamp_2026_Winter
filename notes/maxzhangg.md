## Web3 行业全局介绍 & 岗位概览

### 一、行业阶段与趋势

-   **Web3 四阶段演进**：  
    1）BTC 极客实验期 →  
    2）ETH + ICO 造富期 →  
    3）NFT / DeFi / L2 应用爆发 →  
    4）**合规 + ETF + RWA + DePIN + AI 融合（当前）**
    
-   **2024–2026 核心变化**：监管转暖、BTC/ETH 现货 ETF 通过、机构与传统资本大规模入场，行业从投机走向基础设施与实体结合
    

### 二、就业市场现状

-   **人效极高**：10 人团队可管理数亿–数十亿美金 TVL
    
-   **开发者稀缺**：全球活跃开发者仅约 2.5–3 万
    
-   **2025 招聘反弹**：新增岗位约 6.6 万（+47%）
    
-   **结构性失衡**：
    
    -   初级岗位极度内卷（≈450:1）
        
    -   资深 Rust / 合规 / Research 人才严重短缺
        

### 三、岗位与薪酬梯队

-   **第一梯队**：智能合约 / 安全审计 / 合规（高壁垒 + 高薪）
    
-   **第二梯队**：Research / 核心产品（懂 Tokenomics）
    
-   **第三梯队**：运营 / 增长（普通岗位低，但增长高手天花板高）
    
-   **区域分工**：
    
    -   技术核心：北美
        
    -   交易所 / 运营 / 资管：亚洲 & 中东
        

### 四、求职关键认知

-   **Web3 不看“简历”，看“链上履历”**：  
    GitHub、DAO 治理、Snapshot 投票、Mirror 输出
    
-   **核心渠道**：Web3 垂直平台 + Twitter/X + Discord + DAO
    
-   **必备能力**：
    
    -   Dev：Solidity → Rust / Move / Cairo + 安全意识
        
    -   非技术：英语 + Tokenomics + 链上交互经验
        
-   **避坑原则**：远离“三无项目”（无融资 / 无代码 / 全匿名）
    

### 五、个人反思

Web3 本质是**高杠杆、低人数、强结果导向**的行业，真正的竞争力来自可验证的链上行为与长期输出，而不是传统投递逻辑。
