# web3必装软件

| App 名称 | 在 Web3 里的直白作用 | 一句话大白话 |
| MetaMask | 收钱、发钱、签合同 | 你的钱包+身份证。没它你在 Web3 寸步难行。 |
| Discord | 项目方的“洗脑”和客服基地 | 进群领白名单、蹲空投。每个项目都有自己的大群。 |
| Telegram | 拉小群、传小道消息 | 找项目方私聊，或者在各种“土狗群”里看财富密码。 |
| X (Twitter) | 喊单、吃瓜、看风向 | 大佬们带货和互撕的地方，看现在什么最火。 |
| GitHub | 看这项目是不是在“画饼” | 懂行的人上去看代码更不更新，防止项目方跑路。 |
| Notion | 写白皮书和说明书 | 项目方画饼的草稿纸，告诉你这币以后值多少钱。 |
| LinkedIn | 查查这老板是不是骗子 | 看看项目创始人是不是实名上线，有没有正经履历。 |
