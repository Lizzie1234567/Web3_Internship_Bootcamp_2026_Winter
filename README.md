# Web3 实习计划 - 2026 冬季

## 介绍

「Web3 实习计划」是一个面向高校学生与 Web3 新人的成长型入门项目，由 ETHPanda 与 LXDAO 联合发起，致力于为渴望进入区块链行业的新人提供系统学习、实战训练、岗位对接与社区陪伴，帮助他们跨越信息门槛、打通实践路径，顺利迈出进入 Web3 的第一步。
